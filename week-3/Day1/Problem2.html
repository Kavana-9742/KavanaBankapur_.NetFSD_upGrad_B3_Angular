Frequently Toggle Section

1. What is jQuery?
jQuery is a JavaScript library used for DOM manipulation and event handling.

2. Why use jQuery?
It simplifies JavaScript coding and reduces complexity.

3. What is slideToggle()?
slideToggle() shows or hides content with a sliding animation.

4. What is DOM?
DOM stands for Document Object Model. It represents the HTML structure of a webpage.

5. How to include jQuery?
You can include jQuery using a CDN link inside the script tag.
